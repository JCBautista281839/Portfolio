# Portfolio Landing Page

A professional, modern landing page built with **React**, **Vite**, and **CSS3** featuring smooth animations, responsive design, and a well-organized file structure.

## Project Structure

```
Portfolio/
├── public/
│   └── index.html           # Main HTML entry point
├── src/
│   ├── components/          # Reusable React components
│   │   ├── Header.jsx
│   │   ├── Hero.jsx
│   │   ├── Features.jsx
│   │   ├── Projects.jsx
│   │   ├── Contact.jsx
│   │   └── Footer.jsx
│   ├── pages/               # Page components
│   │   └── LandingPage.jsx
│   ├── styles/              # Component and global styles
│   │   ├── index.css        # Global styles
│   │   ├── Header.css
│   │   ├── Hero.css
│   │   ├── Features.css
│   │   ├── Projects.css
│   │   ├── Contact.css
│   │   └── Footer.css
│   ├── assets/              # Images, fonts, etc.
│   ├── App.jsx              # Main App component
│   └── index.jsx            # React DOM mount point
├── package.json             # Dependencies and scripts
└── vite.config.js          # Vite configuration
```

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will open at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

## Features

✨ **Modern Design** - Gradient backgrounds, smooth animations, glass-morphism effects

📱 **Responsive** - Mobile-first approach with breakpoints for all devices

⚡ **Fast** - Built with Vite for optimized performance

♿ **Accessible** - Semantic HTML and ARIA compliant

🎨 **Customizable** - Easy-to-modify CSS variables for theming

## Color Scheme

- **Primary**: #6366f1 (Indigo)
- **Secondary**: #ec4899 (Pink)
- **Dark Background**: #0f172a
- **Light Text**: #f1f5f9

## Components

- **Header** - Navigation bar with smooth scroll
- **Hero** - Eye-catching landing section
- **Features** - Grid of feature cards
- **Projects** - Project showcase section
- **Contact** - Contact form
- **Footer** - Footer with social links
